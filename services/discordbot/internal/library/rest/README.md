# REST Admin Examples

Comandos:
- `/library-rest-guild`: consulta informações gerais da guild.
- `/library-rest-channels`: lista canais de texto.
- `/library-rest-member`: consulta membro + banimento.
- `/library-rest-roles`: lista cargos.
- `/library-rest-emojis`: lista emojis personalizados.
- `/library-rest-scheduled`: lista eventos agendados.
- `/library-rest-automod`: lista regras de auto moderação.
- `/library-rest-sync-command`: cria/atualiza comando de teste.

Use-os com cautela: alguns endpoints podem exigir permissões elevadas ou modificar recursos (ex.: sync command).
