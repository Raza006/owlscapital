# Messaging Examples

Comandos disponíveis:
- `/library-embed-basic`: embed padronizado.
- `/library-embed-v2`: container/section (embed v2) com componentes.
- `/library-message-edit`: edita/deleta a resposta.
- `/library-attachment-upload`: envia attachment.
- `/library-reaction-manage`: adiciona/remove reações.
- `/library-thread-create`: cria thread e envia mensagem.

Teste cada comando com permissões adequadas. Consulte `docs/library/inventory.md` para cobertura.
