# Gateway Event Examples

Handlers disponíveis:
- Guild/member/role: `guild_events.go`
- Canal/thread/stage: `channel_events.go`, `thread_events.go`, `stage_events.go`
- Mensagens/reações: `message_events.go`, `reaction_events.go`
- Interações, voz, auto-mod, subscription, gateway lifecycle.

Registre a feature desejada via `RegisterLibrary...` antes de iniciar a sessão para observar os logs.
